# 版本变更日志

所有显著的更改都将记录在此文件中。

## [0.0.2] - 2025-07-30

### 新增功能
- **章节完整显示**：移除了分页功能，章节内容现在完整显示，无需翻页
- **章节导航**：新增上一章/下一章快捷键导航功能
  - `Ctrl+Shift+Alt+P` / `Cmd+Shift+Alt+P`：上一章
  - `Ctrl+Shift+Alt+N` / `Cmd+Shift+Alt+N`：下一章

- **章节进度显示**：阅读时显示当前章节进度（如：章节进度: 3/15）
- **自定义文件名显示**：支持为文件设置自定义显示名称
- **文件移除功能**：右键菜单支持移除文件并同步删除配置
- **VS Code设置界面配置**：支持通过VS Code设置界面管理阅读列表文件
- **配置同步功能**：设置界面与JSON配置文件实时同步
- **阅读记录功能**：自动记录最后阅读的章节和行号，在阅读列表中显示阅读进度

### 优化改进
- **阅读体验**：移除了分页控制，阅读更加流畅
- **界面优化**：章节列表更清晰显示章节信息
- **代码重构**：重构了章节导航逻辑，支持完整的章节切换
- **随机代码模式优化**：根据配置百分比，实现每行正文夹杂对应数量掩饰代码的展示逻辑
- **配置管理优化**：完全使用VS Code settings.json配置，移除本地.vs-yuedu-config.json文件依赖
- **图标更新**：新增专业SVG图标，提升插件视觉效果
- **配置简化**：移除旧格式配置兼容代码，统一使用新配置格式

### 修复问题
- 修复了章节切换时的状态管理问题
- 优化了章节识别算法，支持更多格式的章节标题
- 修复了配置文件格式兼容性问题，支持旧格式自动转换
- 增强了文件路径验证和错误处理

## [0.0.1] - 2025-07-30

### 初始版本
- **基础功能**：添加本地文件到阅读列表
- **章节识别**：自动识别文件中的章节标题
- **分页阅读**：支持在终端中分页阅读文件内容
- **侧边栏管理**：树形视图管理文件和章节
- **右键菜单**：支持右键快速添加文件到阅读列表
- **快捷键**：支持分页快捷键操作
- **随机代码模式**：可选的随机代码干扰功能

### 技术特性
- 支持多种章节标题格式识别
- 智能分页算法
- 可配置的随机代码比例
- 完整的VS Code扩展集成
